# Отчёт по работе

Была разработана программа для детекции автомобилей, определения их скорости и времени пересечения линии подсчёта. Для детекции использовалась модель YOLO `yolo26s.pt`, а для отслеживания автомобилей между кадрами использовался ByteTrack. Скорость автомобилей измерялась за счёт подсчёта времени движения на известном расстоянии от линии A до линии B. Для каждого автомобиля программа запоминала время пересечения первой и второй линии, после чего вычисляла скорость.

Формула расчёта:

```text
speed = 3.6 * distance / delta
```

где `distance` - расстояние между линиями в метрах, а `delta` - время между пересечениями линий в секундах.

# Настройка линий

На 13.1 секунде видео был замечен автомобиль Ford S-MAX 2019 года. Длина такого автомобиля составляет примерно 4.8 метра. По началу и концу автомобиля были проведены линии A и B, поэтому расстояние между ними было принято равным 4.8 метров

# Результат

Программа задетектировала и посчитала 55 автомобилей. При ручном просмотре видео было замечено более 70 автомобилей, поэтому часть машин не была обнаружена или не пересекла линию подсчёта с устойчивым ID. Скорость в программе является приблизительной. На неё влияют перспектива видео, примерная длина автомобиля Ford S-MAX 2019 года выпуска, расположение линий, пропуски детектора и возможная смена ID у трекера
